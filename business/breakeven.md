# Breakeven Analysis – Competitor‑Shield

| Section | Details |
|---------|---------|
| **Unit Economics** |  |
| **Cost per Active User** | • **Compute**: 0.0005 USD / hour per user (AWS Lambda 1 ms per inference) → 0.36 USD / month<br>• **Storage**: 0.023 USD / GB‑month (S3) → 0.23 USD / month (average 10 GB per user)<br>• **Bandwidth**: 0.09 USD / GB (outbound) → 0.45 USD / month (average 5 GB per user)<br>• **Total**: **≈ 1.04 USD / month** |
| **Pricing Tiers** | • **Starter** – $12 / mo: 5 AI queries/day, basic dashboard, email support.<br>• **Growth** – $39 / mo: 50 AI queries/day, advanced analytics, API access, priority support.<br>• **Enterprise** – $99 / mo: Unlimited queries, custom integrations, SSO, dedicated account manager. |
| **CAC (Customer Acquisition Cost)** | • **Marketing spend**: $1.2 USD per qualified lead.<br>• **Sales effort**: 0.8 USD per demo.<br>• **Total CAC**: **$2.0 USD** (average). |
| **LTV (Lifetime Value)** | • **Churn**: 5 %/month (average 20 months).<br>• **Average ARPU**: 0.5 × $12 + 0.3 × $39 + 0.2 × $99 = $28.20 / mo.<br>• **LTV** = ARPU × Avg. tenure = $28.20 × 20 = **$564**. |
| **Break‑Even Users** | • **Profit margin**: (ARPU – Cost) / ARPU = (28.20 – 1.04) / 28.20 = 96.3 %.<br>• **Contribution per user**: $27.16 / mo.<br>• **CAC payback**: $2.0 / $27.16 ≈ 0.07 mo (≈ 2 days).<br>• **Break‑even MRR**: $10 k → 10 000 / 28.20 ≈ **355 users**. |
| **Path to $10 k MRR** | • **Starter**: 355 users × $12 = $4,260 / mo.<br>• **Growth**: 355 users × $39 = $13,845 / mo.<br>• **Enterprise**: 355 users × $99 = $35,145 / mo.<br>• **Recommended mix**: 200 Starter, 120 Growth, 35 Enterprise → MRR = 200×12 + 120×39 + 35×99 = **$10,020**. |

**Key Takeaway**  
With a low CAC and high margin, Competitor‑Shield can hit $10 k MRR after acquiring ~355 users, achievable within 3–4 months of launch given the projected 5 % churn and 20 month lifetime.
